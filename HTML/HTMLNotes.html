What is HTML

  • HTML is short for hyper text markup language
  • HTML is not a programming language
  • HTML is a Markup language for creating webpages and documents
  • HTML is a building block of the web

What do you need to get started

  • A webrowser
  • A text editor

Creating an HTML file s

  • Creating an HTML file does not need a server
  • Files must end with .html extension
  • Runs in a web browser
  • Index.html is the root or hompage for the website

Tag Syntax

  • Element names are surrounded by angle brackts
  • Tags normally come in pairs
  • End tag is usually the same as the start tag just with a forward slahs
  • Some tags will close them selves
  • Tags are started using < some tage name > and ended using < some tage name > but with a / before the tag name
  • Tags can be auto completed with by pressing the tab button
  • headers are contained in a h tag there are 6 different header tags h1, h2, h3, h4, h5, h6
  • paragraphs are contained in a p tag
  • some tags are self closing such as br

DOCTYPE

  • The doctype explains what type of document the page is

Inlinve vs block level elements

Inline elements

  • Inline elements do not start on a new line
  • Inline elements take only the necessary width

Block elements

  • Block elements start on a new line
  • Block elements also take the full width availabale.
  • An example of a block level element wold be div, h2, p, form etc
  • An example of a inline level element would be span, img, a, etc

Tag Attributes

  • All tags can have an attribute
  • Attributes provide infromation about an element
  • Attributes are palced within the start tag
  • Atributes are a key value pair
